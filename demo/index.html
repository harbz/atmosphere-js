Web Components Library v1.0

Cinematic texture
for the web

Drop-in web components that add film grain, vignette, light leaks, and more to any element. Zero dependencies. One script tag.

$ npm install atmospherejs View on GitHub →
01 — Grain

Animated Film Grain

Organic Perlin noise that moves like real 35mm film. Apply it to images, hero sections, or entire page backgrounds for instant analog warmth.

Intensity 0.3
Speed
Color
HTML 
<atm-grain intensity="0.3" speed="medium"
  color="neutral" animated="true">
  <img src="photo.jpg" />
</atm-grain>
02 — Vignette

Edge Vignette

The classic darkened-edge effect that draws the eye to the center. Adjust strength, color, and shape to match your scene.

Strength 0.5
Shape
HTML 
<atm-vignette strength="0.5"
  color="#000000" shape="oval">
  <img src="photo.jpg" />
</atm-vignette>
03 — Scanlines

CRT Scanlines

Subtle horizontal scan lines for a retro, analog display aesthetic. Lightweight CSS-only — zero performance impact.

Opacity 0.12
Spacing 3px
HTML 
<atm-scanlines opacity="0.12"
  spacing="3" color="#000000">
  <div>Any content</div>
</atm-scanlines>
04 — Light Leak

Film Light Leaks

Warm light bleed at the corners — the signature look of exposed analog film. Supports breathing animation for a living feel.

Opacity 0.4
Position
Color
HTML 
<atm-light-leak position="top-right"
  color="warm" opacity="0.4"
  animated="true">
  <img src="photo.jpg" />
</atm-light-leak>
05 — Paper

Paper Texture

Scalable parchment and paper textures using SVG turbulence filters. Adds a physical, tactile feel to flat digital surfaces.

A surface that feels real

Like holding aged paper in your hands. Physical. Warm. Grounded in the material world.
Opacity 0.12
Texture
HTML 
<atm-paper texture="medium"
  opacity="0.12" tint="#f5f0e8">
  <div>Content here</div>
</atm-paper>
